# yychoice를 소개합니다!

선택이 고민될 때 사용해보세요!

-   테스트 서버(베타버전): https://yychoice.netlify.app
-   backend repository: https://github.com/kimyoungyin/yychoice__server

## 스택

-   client: Typescript, React, Axios
-   server: Typescript, Express
-   typeORM: Sequelize
-   database: mysql, RDS
-   storage: S3
-   deploy: netlify(client) + ec2(server)
